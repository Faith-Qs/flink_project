# bigdata_analyse
大数据分析项目

## wish

采用不同的技术栈，通过对不同行业的数据集进行分析，期望达到以下目标：

- 了解不同领域的业务分析指标 
- 深化数据处理、数据分析、数据可视化能力
- 增加大数据批处理、流处理的实践经验
- 增加数据挖掘的实践经验
